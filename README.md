# Librería de Números Complejos en Python

Este proyecto implementa una librería en Python para realizar operaciones con números complejos sin utilizar el tipo de dato `complex` incorporado en el lenguaje.

Los números complejos se representan mediante tuplas:
- Forma cartesiana: (parte real, parte imaginaria)
- Forma polar: (módulo, fase)

# Features

La librería soporta las siguientes operaciones con números complejos:

- Suma

- Resta

- Producto

- División

- Módulo

- Conjugado

- Fase

- Conversión entre coordenadas cartesianas y polares
